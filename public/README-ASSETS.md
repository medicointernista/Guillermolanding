# Assets Needed

Para que la landing page funcione completamente, necesitas agregar las siguientes imágenes en la carpeta `public/`:

## Imágenes requeridas:

1. **manejo-del-dolor-ortopedico-en-medellin.png** - Logo principal (tamaño recomendado: 200x200px)
2. **hero-doctor.png** - Imagen del doctor en la sección hero (tamaño recomendado: 600x750px, aspect ratio 4:5)
3. **dr-guillermo-rodriguez.jpg** - Foto del doctor para la sección "Sobre el médico" (tamaño recomendado: 480x640px, aspect ratio 3:4)
4. **logo-sccot.jpg** - Logo de SCCOT
5. **logo-accart.jpg** - Logo de ACCART
6. **logo-andes.jpg** - Logo Universidad de los Andes
7. **logo-ces.png** - Logo Universidad CES

## Ubicación:
Todas las imágenes deben estar en la carpeta `/public/` del proyecto.

## Nota:
Actualmente las rutas de las imágenes están configuradas para buscar en `/public/`, que en producción se sirven desde la raíz del dominio.
